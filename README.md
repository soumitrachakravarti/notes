# Notes
A CLI application for taking notes

# Usage
git clone

yarn install

node index.js --help

# Example

## Adding a note
node index.js add --title="Note tilte" --body="Note body"

## Reading a note
node index.js read --title="Note tilte"

## Removing a note
node index.js remove --title="Note tilte"

## Listing all notes
node index.js list
